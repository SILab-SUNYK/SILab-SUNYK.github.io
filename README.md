# SIL — Spatial Intelligence Lab

A lightweight Jekyll site designed for GitHub Pages.

- Edit publications in `_data/publications.yml`. Add an optional `image` and `image_alt`; store thumbnails in `assets/images/publications/`.
- Edit people in `_data/team.yml`
- Edit news in `_data/news.yml`; each item supports optional `image`, `image_alt`, `details`, and `link` fields
- Replace homepage images in `assets/images/home/` and update their paths in `_data/home.yml`
- Edit homepage content in `index.html`

Run locally with `bundle install` and `bundle exec jekyll serve`.
